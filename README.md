# ShadowTrace AI

Explainable Behavioral Cybercrime Intelligence for Coordinated Identity Detection

## Overview
ShadowTrace AI is a SOCMINT prototype designed to assist cybercrime investigators in identifying coordinated digital identities using behavioral fingerprinting and Explainable AI.

## Features
- Behavioral similarity scoring
- Stylometric analysis
- Relationship graph visualisation
- Explainable AI reasoning
- Deterministic scoring pipeline
- Downloadable investigation reports

## Technical Stack
- HTML/CSS/JavaScript
- Behavioral scoring heuristics
- NetworkX-inspired graph logic
- Explainable AI workflow
- OSINT-based dataset simulation

## Compliance
- CERT-In aligned
- IT Act 2000 aware
- Public OSINT-only analysis

## Prototype
Live demo available via GitHub Pages / Netlify deployment.

## Disclaimer
Outputs are investigative leads only and require human review before escalation.
